# covergates
Covergates CLI for NodeJS

## Installation

```sh
npm install covergates --save-dev
```

## Usage

`covergates` will install a binary under `node_modules/.bin/covergates`.
To use it under CLI, run:

> For npm v5.2.0 or newer
```sh
npx covergates -h
```

or

```
./node_modules/.bin/covergates
```

For the `covergates` command line usage, please refer to the [documentation](https://docs.covergates.com/start/cli/).

## Example

You can find the example for NodeJS test coverage with covergates at [https://github.com/covergates/example-nodejs](https://github.com/covergates/example-nodejs).
